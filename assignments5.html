Any of the remaining four can sit in the third chair, and any of the remaining three can sit in the fourth chair. (The last two will have to stand.) So the number of possible arrangements of six guests in four chairs is 6*5*4*3, which is 360.         cin >> noOf_chairs;

    int upper = noOf_Guest;

Task 9. A prime number is a number that is only evenly divisible by itself and 1. For example, the number 5 is prime because it can only be evenly divided by 1 and 5. The number 6, however, is not prime because it can be divided evenly by 1, 2, 3, and 6. Write a program, which takes an integer from user and print “Number is Prime” if the number is a prime number, or “Number is not Prime” otherwise.

    cin>>num;
    while(i<num)
    {
        if(num%i==0)

            flag=false;
        }

    {
